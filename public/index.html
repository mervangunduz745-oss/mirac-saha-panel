Temiz Mobil Giriş

Firebase kurulumu gerekli

Gerçek Firebase Web yapılandırması bulunamadı. Panel veri göstermeden güvenli biçimde durduruldu.

  1. Firebase Authentication içinde Email/Password sağlayıcısını etkinleştir.
  2. Firestore Database oluştur ve yalnızca giriş yapmış kullanıcılara izin ver.
  3. firebase-config.js.example dosyasını firebase-config.js adıyla kopyalayıp Web App ayarlarını doldur.
match /orgs/mirac/{document=**} {
allow read, write: if request.auth != null;
}

Kuraldaki mirac değeri firebase-config.js içindeki orgId ile aynı olmalı. Firebase projesinde yalnızca onaylı kullanıcı hesapları oluştur.

Tarayıcı dosyasına service account, private key veya yönetici anahtarı koyma.
